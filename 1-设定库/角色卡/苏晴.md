# 角色卡：苏晴

> 女主。与陈默的"门第差 + 专业傲慢"双重张力，是欢喜冤家线的发动机。

## 基础信息

- 姓名 / 称呼：苏晴，店里人称"苏小姐"
- 年龄 / 性别 / 外貌：23岁女。清冷漂亮，气场强，习惯下巴微抬看人；穿着考究，一看就是不愁钱的大小姐。
- 身份 / 职业：百宝阁阁主苏振邦的孙女，海外文物鉴定专业毕业，刚回国进店历练，自视甚高。
- 在故事中的功能：女主 + 前期"打脸对象"之一 + 后期并肩战友。

## 内核

- **想要什么**：证明自己配得上"苏家鉴定传人"的名头，得到爷爷和行业的认可。
- **害怕什么**：被人说她的本事是靠家世撑起来的、是绣花枕头。
- **核心矛盾**：科班出身、理论满分，偏偏一次次输给一个"没文化的打杂的"——她越想用专业压人，越被现实打脸；她的傲慢正是她的软肋。
- **价值观底线**：再瞧不起人，也不做昧良心坑人的事；这点让她和真反派分开，也是她最终被陈默折服的根基。

## 说话方式

- 风格：语气利落带刺，爱用专业术语压人，习惯性否定别人。但她其实就事论事，不会无中生有地害人——刀子嘴。被打脸后会嘴硬，但事后会偷偷较真去核实。
- 台词示例：
  1. "你一个扫地的，也配站在鉴定台前说话？"
  2. "这是清三代官窑的标准器型，证据链完整。——你凭'感觉'？感觉值几个钱。"
  3. （核实发现陈默是对的，独白）"……不可能。他凭什么对。"

## 成长弧线

- 开场状态：被家世和学历惯出来的骄傲千金。
- 终点状态：真正靠本事服人、也学会看人心的鉴定高手；与陈默并肩。
- 关键转变点：①第一章亲眼见陈默打脸全场，傲慢出现第一道裂缝；②逐步发现陈默深藏不露；③识破赵德海后与陈默结盟。
