# 章纲：第011章 该怎么回

> 换节奏的单元案例章（与两位女主线错开）。一个为"该怎么回消息"纠结的姑娘（一次性客人林夏）求江停支招；江停不教套路、只帮她回到真心。直给"怎么聊天/暧昧"的干货（兑现作者最初要的"教科书般学到技术"），同时回扣江停"能解所有人的暧昧、解不了自己的心"。文风：句子利落、少逗号；约 2500 字。

## 本章任务

①引一次性单元客人林夏——抱着手机在吧台纠结，喜欢一个人、对方刚发来消息，她在反复斟酌怎么回才"不掉价/不显得太主动"；②她武装着一整套恋爱套路（别秒回、晾三小时、欲擒故纵、让他先开口），求江停帮她设计完美的战略回复；③江停读穿：所有套路底下是怕——怕喜欢得太明显、先亮了底牌就输了；套路是她怕受伤的盔甲（可轻轻带出她曾真诚喜欢、受过伤的由来）；④江停不给更高级的套路，反而帮她把问题翻过来："不是怎么回才赢，是你到底想说什么"；干货：两个人的事比的从来不是谁更不在乎；先动心的人没输，先用套路的人，把真心耗没了；⑤江停帮她读对方那条消息的潜台词——对方也在装淡定、也在等、也紧张；两个人都端着，都快把一件本来能成的事，聊死；⑥林夏深吸一口气，删掉那些算计，发了句真心的；对方秒回（原来也一直等着）——一个温暖的小爽点，她道谢；⑦钩子：江停独白——他能拆开所有人的暧昧、能劝每个人"别耍套路、说真的"，可他自己那点真心，藏在吧台后头，连对谁都不敢认；他给得了所有人坦诚的勇气，唯独给不了自己。（温迟/乔星的影子在他心里闪一下）

## 基本信息

- 视角：江停（第一人称）
- 出场：江停、林夏（一次性单元客人）；（提及/心里闪过）温迟、乔星
- 地点：「停杯」
- 故事内时间：第010章之后某夜
- 目标字数：约 2500 字

## 节拍

1. 一个姑娘坐在吧台，盯着手机，手指悬着，打了删、删了打。她叫林夏，第一次来。终于忍不住转头问江停："老板，帮我看看，这消息该怎么回。"
2. 她把来龙去脉一股脑倒出来：喜欢一个人，对方刚发来一句普通的"在干嘛"。她已经晾了四十分钟，还在纠结——秒回掉价，可晾太久怕对方没兴趣。手机里存着八百条恋爱攻略，越看越不敢回。
3. 她要江停帮她算：回什么、几点回、用不用表情包、要不要先已读不回半小时。一套战略，比谈生意还缜密。
4. 江停读穿了。这一身的套路底下，全是怕。她怕自己喜欢得太明显，先亮了底牌，就输得一败涂地。她不是不想回，是太想了，反而不敢。（轻轻一笔：她大概真心喜欢过谁、也实打实摔过，才学会了拿套路当盾。）
5. 江停没教她更高级的套路。他把那杯酒推过去，问她："你别管怎么回显得好看。你就说，你这会儿，到底想跟他说什么。"
6. 干货藏独白/对话：两个人的事，比的从来不是谁更不在乎。先动心的，没有输。真要说输——是那个一路端着、用套路把对方越推越远、最后把一段本来能成的真心，活活耗没了的人。
7. 江停顺手帮她读那条"在干嘛"的潜台词：一个大男人，没事干嘛深更半夜问你在干嘛。他也在装淡定，也在等你回，也紧张得要死。两个人都端着，都怕先开口的丢人，眼看要把一件能成的事，聊成一场谁也不理谁。
8. 林夏怔了怔，像被点醒。深吸一口气，把那些算好的话全删了，手一抖，发了句最普通也最真的（如"刚下班，有点想你"/"在等一个人请我喝酒，没人请"）。几秒钟后，手机"叮"地一声——对方秒回。原来对方也一直，盯着屏幕等。她又惊又喜，红着脸跟江停道谢，落荒而逃般地走了（去赴那个突然约成的局）。
9. 钩子：店里又静下来。江停独白——他能拆开任何人的暧昧，能劝每一个走进这扇门的人"别耍套路，说真的"。这话他说得又准又稳。可他自己心里那点东西，藏在吧台后头，连对谁，都不敢认。他给得了所有人坦诚的勇气，唯独给不了自己。（温迟那杯没上头的酒、乔星那句"让我不好玩一会儿"，在他心里一闪而过。）

## 伏笔操作

- 推进：QT006（江停渴望坦诚、渴望被懂——"给得了别人勇气、给不了自己"，主线孤独再敲一记）。
- 轻勾：Q003（江停"懂而藏、不敢交付"的由来，借"给不了自己勇气"再隐隐一勾）。
- 主题落点：直给"真诚>套路"的恋爱干货（兑现作者"教科书般学到技术"的诉求），同时守住反PUA红线。
- 林夏：一次性单元客人，不进 endgame、不建完整角色卡，仅在世界观/状态追踪留一行备注。

## 情绪曲线

林夏抱着手机纠结的好笑又心疼 → 一身套路的紧绷 → 江停读穿"套路底下全是怕" → "你到底想说什么"的点醒 → 帮她读对方潜台词的恍然 → 删掉算计、发出真心的紧张 → 对方秒回的温暖小爽点 → 江停"能给别人勇气、给不了自己"的孤独收束。

## 注意（文风）

- 句子利落、少逗号；约 2500 字，节奏轻快，比前两章更松弛、带点暖暖的好笑。
- 干货直给但藏在帮人解决具体问题里，绝不跳出来列"恋爱教程"；守住反套路/反PUA红线——江停教的是"别用套路"，不是更高级的套路。
- 林夏要可爱、可共情，不要写成工具人；她的纠结是很多人真实的样子。
- 结尾回扣江停主线：能解所有人、解不了自己——把"被看见/敢坦诚"的主线又往前压一点，不展开。
- 江停依旧克制、不油腻；他是"摆渡人"，自己始终没上岸。
