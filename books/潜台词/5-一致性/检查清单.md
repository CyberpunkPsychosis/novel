# 章节自查清单

> AI 每写完一章对照自查。任何一项不通过，先修正再交稿。

## 一致性

- [ ] 出场角色的言行符合角色卡（尤其是说话方式和价值观底线）
- [ ] 没有引入设定库之外的重要设定
- [ ] 故事内时间与时间线衔接无矛盾（季节、昼夜、相隔天数）
- [ ] 角色的知情范围正确——没有人说出他不该知道的信息
- [ ] 持有物、伤势与状态追踪表一致

## 结构

- [ ] 覆盖了章纲的全部节拍，没有擅自增删重要剧情
- [ ] 章末落在钩子上
- [ ] 本章的伏笔埋设/回收已按章纲执行

## 文风

- [ ] 视角纪律未违反
- [ ] 没有踩文风指南的雷区
- [ ] 字数在目标范围内（±20%）

## 收尾动作

- [ ] 已更新 `状态追踪.md`（角色状态、伏笔账本、钩子清单、当前进度）
- [ ] 已更新 `时间线.md` 正文时间线
- [ ] 重要角色塑造已回写角色卡「经历更新」
