# 故事核

> 完结导入稿。本书为已完成的长篇，按仓库规范归档、上架；此文件存其内核要点，供日后续写番外或修订参考。

## 一句话故事

刚被裁员的陆时手机里凭空多出一个名叫「一面」的 App，每天零点推送一个"今日匹配"、限他当天只见一面；从替他点酒、一眼叫破他名字的乔棠开始，一面接一面的相遇，把他一步步领向三年前那个他不敢想起的夜晚的真相。

## 类型 / 体量

- 类型：都市悬疑 / 爱情 / 反转 / 暗黑系。
- 体量：长篇，12 章 + 尾声，已完结。

## 主题

- 母题：**一面之缘**与宿命——每天只够见一面的残酷规则背后，是一桩被刻意遗忘的旧案与一份未了的执念。
- 底色：不敢争、被按了暂停键的人，被"一面一面"逼着重新面对过去、夺回自己的人生。

## 核心设定

- 「一面」App：无来由地出现，每天零点一个匹配（照片+地点+一句话），"迟到的人没有第二次"，每天只允一面。
- 悬疑钩：乔棠仿佛早就认识陆时；匹配并非艳遇，而是指向三年前那个夜晚的真相与一桩旧案。

## 主要人物

- 陆时：被裁员的前策划组长，男主，第一人称视角。
- 乔棠：神秘女子，每日匹配的起点，知陆时甚深。
- 裴谦：反派（尾声"裴谦的帝国塌了"），与旧案相关。
