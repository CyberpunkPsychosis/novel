# 章纲：第024章 成全

> 章纲是作者和 AI 对齐的合同：正文必须覆盖所有节拍，不得擅自增删重要剧情。

## 本章任务

收束顾深线，给男二一个有尊严、有温度的退场；同时以"破局程序已启动、进入煎熬的等待期"作背景，让苏念清清楚楚地选定陆砚、坦诚告知顾深。顾深成全、退场体面，甚至以己之力为陆砚后路搭把手——兑现其"清醒、善良、不占便宜"的人设，反衬陆砟"不说"的代价，也为后段二人重圆清场。

## 基本信息

- 视角角色：苏念
- 出场角色：苏念、顾深；（提及/背景）陆砚、苏父、周明
- 地点：城里某处（顾深约见 / 苏念赴约）
- 故事内时间：第023章后约一两周（初冬，程序已启动、等待期）
- 目标字数：不设硬指标，写透节拍即可

## 节拍（按顺序）

1. 【开场·背景交代】这一两周，父亲已随周明把该走的程序起了头（自首/申诉，司法模糊化）。进入漫长而煎熬的等待。苏念与陆砚，第一次以"并肩"的姿态一起面对——背景一笔带过，重心转向她欠顾深的那个交代。
2. 苏念约见顾深（或顾深约她）。她知道，欠他一个清楚的答案，再难也得当面给。
3. 她坦诚相告：她选定了陆砚，要陪他把这条路走到底；顾深的好，她记一辈子，但她不能给他想要的回应。说得诚实、郑重，不留暧昧的尾巴。
4. 顾深的体面退场：他其实早看出来了（自第012章）。他不闹、不怨、不卖惨，带着一点自嘲的洒脱接受。给一两句他特有的"会好好说话"的收束（如"好东西都得慢慢来，只可惜，没能等到我这桌"）。守住尊严与温度。
5. 成全到底：顾深不让未得的情，变成小气。他主动提出，以自己的人脉/路子，为陆砚日后重新立足搭把手（或只是一句真诚的祝福与"有需要尽管开口"）。兑现人设，也反衬陆砟"宁可烂着也不说"的代价——会好好说话、肯把话摊开的人，连退场都干净利落。
6. 【章末钩子】两人好聚好散。苏念目送顾深离开，心里有歉、有敬、也有一种尘埃落定的轻。她转身，前面是更难的仗（父亲的程序结果未卜、陆砟的清白未还），可她的心，从未这样定过——该了断的了断了，该并肩的并肩着。她抬头望向陆砚那边（电话/归途），一句在心里：接下来，就剩我们一起，把这条路，走完。（落在"清场后全心向前"的笃定上。）

## 本章伏笔操作

- 收束：顾深线干净退场（呼应第003、007、012章其人设基线），为二人重圆清场；其"睁眼相助/成全"再次反衬陆砟"不说"之过这一主题对照。
- 背景推进：破局程序已启动、进入等待期（自首/申诉，模糊化），为第025+章的结果与重圆铺垫。
- 不新增谜题；本章为情感收束与清场。

## 情绪曲线

等待期的沉与定 → 赴约的郑重与歉意 → 坦诚相告的不舍 → 顾深体面退场的酸与暖 → 被其成全打动的敬意 → 尘埃落定、全心向前的轻与笃定（落在"剩我们一起走完"上）。
