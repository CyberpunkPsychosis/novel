# 章纲：第018章 红布下面

## 本章任务

推进"父亲那把钥匙"并完成 F006（红布盖物）的揭晓：苏念回老宅，不审判、而是让父亲看见他的"保护"把陆砚害成了什么，温柔却坚定地提出"凿出口需要您肯面对"；父亲在赎罪与恐惧间煎熬；父亲亲手揭开红布，露出他日日上香之物——陆砚的旧物与一封当年的信。信中带出新的一层：陆砚是父亲一手提携、视如己出的孤儿，顶罪既为护苏念、也为报父亲的恩——把死结的情义底色加深。苏念尊重父亲的选择权（不逼他、不替他决定，这正是她与陆砟之别），把"是否站出来"留给父亲自己。

- 视角角色：苏念
- 出场角色：苏念、苏明远；（信中/回忆带出）陆砚
- 地点：苏家老宅
- 故事内时间：第017章后一两日（深秋）

## 节拍（按顺序）

1. 【开场】苏念回老宅。父女之间已无秘密，却横着更难堪的沉默——父亲羞愧得不敢看她。她没有审判，先坐下来，像对待一件极碎的器物，慢慢靠近。
2. 她把陆砚如今的样子讲给父亲听：清苦独居、吃着胃药硬撑、把真相锁死、"宁可烂在里头"。她让父亲清楚地看见，他这三年的"被保护"，把那个孩子，熬成了什么。
3. 她提出请求（温柔而坚定）：周律师说也许有路，但绕不开您肯不肯面对。"爸，您欠他的，不是一辈子的愧疚和上香——是让他不必再替您背下去。"
4. 父亲的煎熬：恐惧（年迈、牢狱、身败名裂），更深的怕是——怕苏念从此把他看成"毁了她丈夫和婚姻的人"，怕失去女儿。赎罪之心与自保之惧在他身上撕扯。
5. 转折与揭晓：父亲终于颤抖着，亲手揭开了那块红布。匣中是陆砚被吊销的旧证、几样旧物，和一封陆砟当年留给父亲的信。信里带出新的一层——陆砟是父亲早年一手提携、近乎视如己出的孤儿；当年顶罪，既为护苏念，也为报父亲多年如父的恩。苏念读着，五内俱裂：这死结，是两代人的情与债，缠死在一处。
6. 【章末钩子】苏念没有逼父亲当场表态。她想起自己最恨的就是"被人替着做决定"——她不会对父亲做同样的事。她把选择，郑重地交还给父亲："爸，我不替您决定。但我求您，别再用沉默，替陆砚决定他该烂一辈子。"她起身离开，把那道最沉的门，留给父亲自己推。而父亲独自坐在揭开了红布的供桌前，第一次，没有重新盖上那块布——一个细小却分明的松动。

## 本章伏笔操作

- 回收：F006 红布盖物正式揭开（内为陆砚旧物+当年信）。
- 新立/加深：陆砚身世——父亲一手提携、视如己出的孤儿；顶罪兼有"报恩"底色（深化动机与死结的情义重量）。须补入设定库（世界观/陆砚、苏明远角色卡）。
- 推进：父亲那把"肯不肯站出来"的钥匙松动一线（结尾不盖红布的细节），但不当场承诺，保留张力。
- 主题强化：苏念"把选择权交还父亲"，正面体现"不替所爱之人做决定"，与陆砟形成对照、也为陆砟弧线作镜。

## 情绪曲线

难堪的沉默 → 让父亲看清代价的沉痛 → 提出请求的克制坚定 → 父亲煎熬的揪心 → 揭红布、读信时的五内俱裂 → 交还选择权的郑重与不舍（落在父亲"没有重新盖布"的微小松动上）。
