# 角色卡：顾深

## 基础信息

- 姓名 / 称呼：顾深。苏念称"顾先生"。
- 年龄 / 性别 / 外貌特征：34 岁，男。温文、得体，笑起来让人放松；古瓷收藏爱好者/相关行业人士。
- 身份 / 职业：藏家 / 美术馆或拍行相关（待定稿），是苏念工作室的重要客户兼朋友，对她有意。
- 在故事中的功能：男二 / "安全选项"。提供与陆砚截然相反的相处方式——坦荡、主动、好好说话——既制造拉扯，也反衬陆砚的"不说"之过。**非工具人反派**，他是真诚体面的，败也败得体面。

## 内核

- **想要什么**：光明正大地追到苏念，给她安稳。
- **害怕什么**：看出她心里那道没补好的旧裂缝，自己始终进不去。
- **核心矛盾**：他什么都对——会说、会等、会尊重——唯独不是她放不下的那个人。
- 价值观底线：不趁虚、不诋毁陆砚、不逼苏念；输了会干净退场。

## 说话方式

- 风格：得体、坦率、有分寸的幽默。该表白表白，该退让退让，从不让人为难。
- 台词示例：
  1."我不急。好东西都得慢慢来——你修瓷不也这么说？"
  2."他要是对你说一句实话，可能就没我什么事了。可他没说，是不是？"

## 经历更新（每章回写）

- 第003章：首次登场。到工作室送一只家藏清三代豇豆红镯子给苏念作参照（暂存她处），递咖啡、闲谈。看出她不在状态，体贴点破却不逼问，问一句被婉拒即止步；临走轻轻替她说句公道话（"谁让你一整天缓不过来，挺不识好歹的"）。坐实其"坦荡、会好好说话、不催不挤"的人设基线，与陆砚"不说"成镜像。尚不知委托人是陆砚、不知前情。
- 第007章：取回镯子。一眼看穿苏念在追"一个没放下的人"，坦然说破却毫不勉强，选择睁着眼帮忙——"你心里有没有别人，和你要我帮忙，是两回事，我分得清"。当场托人问得线索（周律师）递给她，并点出"那件事比你想的还沉"。立住其清醒、有尊严、不占便宜的深度（非舔狗），三角张力拉满。
- 第012章：查破"她要找的人=前夫陆砚"，与苏念坦诚摊牌：坦白心意、自陈"不是圣人"、不要挟不逼；被她诚实拒绝后只道"宁可你这么实在地拒绝，也不要你为难自己迁就我"。仍把查到的硬事实（事故出苏父公司项目、陆砟认责）交给她——"帮你和喜不喜欢你是两回事"。人设升华：清醒、体面、善良的退一步。
- 第024章【退场】：被苏念当面诚实拒绝（她选了陆砟），体面洒脱接受、不闹不怨（"好东西都得慢慢来，只可惜没等到我这桌"）；成全到底——主动允诺以人脉助陆砟日后重新立足，临别讨"一顿喜酒"。线收束。其"会好好说话、肯摊开、退得干净"正面反衬陆砟"不说"之过。
- 第028章（终章·尾声）：体面地续做朋友；如约来喝喜酒，席间笑闹讨红包，临别拍陆砟肩"她就交给你了，对她不好我可不答应"，洒脱离去。圆满收束。【终】
