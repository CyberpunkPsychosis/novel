# 故事核

> 这是全书的最高依据。任何剧情、角色、文风上的分歧，最终都回到这里裁决。
> 当前为初稿，待作者确认后定稿。

## 一句话故事

做古瓷金缮修复的苏念，三年前被前夫陆砚一手"凉薄"逼着签了离婚协议、远走京都；三年后她带着事业和一身锋利归国，被迫与他重新交集，却一层层挖出——当年那场让她心死、让她家差点万劫不复的变故里，陆砚的薄情全是装的，他替她父亲、也替她，独自背下了一口足以毁掉两个家的黑锅，宁可被她恨一辈子，也没解释一个字。

## 类型与体量

- 类型：现代都市 / 言情 / 破镜重圆（虐恋后甜）。女主视角。
- 预计体量：都市言情长篇连载，单章约 2800–3200 字。第一卷「重逢」约 25–30 章先走完"被迫靠近 + 第一道裂缝"的完整弧线，整体按长篇规划，可续。
- 平台定位：都市 → 都市情感 / 言情。对标"破镜重圆""前夫文""暗恋成真"标签读者。

## 主题

- 主线情感引擎：**破镜重圆**——不是回到没碎之前，而是带着那道"金线"（真相与伤痕）重新长在一起。
- 母题（贯穿意象）：**金缮**。把摔碎的瓷用大漆和金粉一片片补回，裂缝不藏、用金线留着，碎得越狠补好越美。女主修的每一件碎瓷，都在暗暗对应她和他这段关系的进度。
- 底色：**真正的深情，有时是甘愿被误解。** 但这不是替"不解释"洗白——全书也要让女主把"你凭什么替我做决定"这句话狠狠还给他。重圆是双向的：他要学会说，她要学会信，谁也不欠谁。

## 目标读者与阅读体验

写给爱看都市言情、尤其吃"破镜重圆 / 误会清醒 / 暗恋成真"这一口的读者。翻页动力按优先级：

1. **真相钩子**——"他当年到底为什么装不爱？"这条悬念贯穿全书，每揭开一层就反转一次读者对前情的理解；
2. **重逢拉扯**——女主带着恨与防备、男主带着隐忍与克制，针尖对麦芒的对峙与心动的暗涌，越克制越上头；
3. **女主爽感**——她不是回来求和的。她带着事业、底气和锋利回来，价码翻三倍他照签；重圆建立在她重新站稳之后的"我配得上被好好爱"，而非单方面原谅。

## 核心设定 / 卖点（替代"金手指"）

本书没有超能力金手指，卖点是**情感结构 + 职业意象**：

- **金缮修复师**：女主是国内少见的古瓷金缮匠人，京都进修归来。职业贯穿全书——碎瓷的拼合进度 = 关系的修复进度；一件反复修不好的器物会成为关键道具（见伏笔）。
- **顶罪反转**：男主三年前主动顶下本属女主父亲的一桩重大责任事故（详见下"前情真相"），并刻意装出薄情凉薄，逼女主离开、与他撇清。这是全书最大的悬念与情感炸点。
- **三层信息差**：读者、女主、男主三方知情度不同步。女主从"全知道（其实全错）"逐步逼近真相，是本书最核心的叙事张力。

## 前情真相（全书命门，前期严格保密，仅作者与 AI 知道）

- 女主父亲**苏明远**经营一家古建筑修复 / 加固工程公司。三年前，一处他亲手签批的工程出了重大质量事故（结构隐患被掩盖、险些酿成人命），按法律与行业责任，足以让苏明远身败名裂、入狱、公司清盘，苏家彻底完蛋。
- 当年的陆砚是苏明远最器重的年轻人、也是苏念新婚不久的丈夫。他查清隐患真正源头牵连复杂、且铁证会把苏父钉死后，**主动把全部责任揽到自己身上**——伪造成"是他擅自篡改了检测数据、是他冒进"，替苏明远顶下罪与债。
- 为了让苏念干脆地离开（不被一个"罪人"拖累、也不卷进案子、更不因心软留下而毁掉前程），他**刻意装出薄情**：冷暴力、推开、甚至默许一段"他变心"的假象，逼她在最痛时签字离婚。
- 代价：陆砚背了刑责/巨额赔偿/行业封杀（具体程度待定稿），三年里独自扛着，从云端跌落又悄悄重建。苏明远知情，但被陆砚以"你要是认，念念这辈子就毁了"按住，三年来负罪缄默。
- 现在（三年后）：陆砚刑期已满 / 风波沉底，低调重新立足；苏念学成归国创业，被一桩生意（他名下旧宅 / 旧藏品的修复委托）逼着与他重新交集。真相从一件碎瓷、一行旧字迹开始裂开。

## 基调与边界

- 虐而有度：前中段拉扯、误会、痛感要够，但不堆无意义的窝囊气；女主始终"立得住、走得动"，不恋爱脑、不卑微。
- 反转要"早有伏笔、回看成立"：每一次真相推进，前文都能找到可被重新解读的细节。
- 甜要"补在裂缝上"：和解后的甜，质感来自"知道彼此最难看的那一面之后，依然选择对方"，不是失忆式重来。
- 不洗白"不解释"：男主的隐忍是深情，也是傲慢的越界。女主清醒后必须把这一点掰扯清楚，二人是在"重新认识彼此"的基础上复合，而非女主单方面原谅。
