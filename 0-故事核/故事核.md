# 故事核

> 这是全书的最高依据。任何剧情、角色、文风上的分歧，最终都回到这里裁决。
> 当前为初稿，待作者确认后定稿。

## 一句话故事

一个在古玩城打杂、被所有人当透明人的穷小子陈默，因一次意外得到能看穿万物的「法眼」，从此一路捡漏鉴宝、扮猪吃虎，在尔虞我诈的古玩与赌石江湖里翻身崛起——可他越往上走，越发现这双眼睛的来历，和父亲当年那桩没说清的旧案缠在一起。

## 类型与体量

- 类型：都市 / 鉴宝赌石 / 扮猪吃虎爽文
- 预计体量：网文长篇连载，单章约 2800-3200 字。第一卷「古玩城崛起」约 25-30 章先写出一个完整的"翻身"弧线，整体按百万字级规划，可续。
- 连载平台：（待定，先按番茄/起点那类节奏写）

## 主题

- 主线爽感：**被低估的人，亮出底牌的那一刻**。
- 底色（副）：眼睛能看穿宝物的真假，却看不穿人心——主角一路要学的，是分辨比古董更难鉴的"人"。

## 目标读者与阅读体验

写给爱看都市爽文的读者。翻页动力按优先级：
1. **打脸解气**——权威/阔少/黑心老板看扁主角，被一一打脸；
2. **捡漏暴富**——平平无奇的破烂里看出国宝，一夜翻身的财富爽；
3. **悬念钩子**——法眼之谜 + 父亲旧案这条长线，让爽点之外有"追更"的劲。

## 金手指：法眼

- 能力：左眼成「法眼」，能透视物体内部、看穿古董真伪与年代、看赌石玉料的成色走向。
- 成长性：每鉴定有"灵气"的宝物会积累眼力，法眼逐级进阶——先能透视，进阶后能看物件的"气运/流转",后期甚至能看人体病灶、看人的情绪起伏（给长线扩展留口子，前期只放透视鉴宝）。
- 代价/限制：用眼极耗精神，过度使用会剧烈头痛、短暂失明——给能力上锁，防止无脑开挂，制造紧张感。

## 扮猪吃虎的三层理由（人设地基，防"装低调"显假）

1. **外部**：法眼太逆天，一旦暴露，古玩界、黑市、乃至更大的势力都会来夺眼觊觎——他必须藏。
2. **性格**：陈默本就是沉稳、不爱出风头的性子，吃过苦，只想先闷声发财把日子过起来。
3. **动机（长线）**：法眼的来历指向父亲早逝那桩旧案，他需要低调潜行、慢慢查清——树大招风会打草惊蛇。

## 打脸燃料（稳定的"看不起他的人"来源）

古玩城「百宝阁」打杂小工的身份 → 周围全是把他当透明人的同行、自封权威的"鉴定专家"、来寻宝装阔的富二代、克扣工钱的黑心老板。主角以"打杂的"身份一次次捡漏、断真伪、赌涨石，反差打脸。

## 主线走向（粗线条）

打杂小子 → 凭法眼捡漏立威、攒下第一桶金 → 自立门户 → 古玩鉴宝界新星 → 卷入文物黑市/盗墓/拍卖局的更大棋局 → 父亲旧案与法眼之谜逐步揭开。

## 结局的大方向

偏圆满。陈默成为一代鉴宝宗师，查清父亲旧案、守住底线没有黑化；财富、地位、感情都有归宿，但为这双眼睛也付出过代价（暂留悬念，写到中段再定细节）。

## 红线（绝对不能发生的事）

1. 主角不碰真正伤天害理的事（不杀无辜、不害好人），打脸对象限定为"该打的"。
2. 法眼不能万能——必须守住"耗精神、会失明"的代价，不能想看就看、看啥都准。
3. 不靠贬低女性/开后宫低俗桥段制造爽点（感情线怎么处理待作者定）。
